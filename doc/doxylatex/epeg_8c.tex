Return image pixels in the decoded format from the specified location rectangle bounded with the box {\ttfamily x}, {\ttfamily y} {\ttfamily w} X {\ttfamily y}. Return image pixels in the decoded format from the specified location rectangle bounded with the box {\ttfamily x}, {\ttfamily y} {\ttfamily w} X {\ttfamily y}. The pixel block is packed with no row padding, and it organsied from top-\/left to bottom right, row by row. You must free the pixel block using \hyperlink{epeg_8c_adf9efc5d877afebda99aba8d5c2bbb0f}{epeg\+\_\+pixels\+\_\+free()} before you close the image handle, and assume the pixels to be read-\/only memory.
